# saucedemo-automa# Saucedemo Automation QA

Proyecto de automatización de pruebas end-to-end usando Playwright y JavaScript.

## ¿Qué automatiza este proyecto?
- Login exitoso
- Login con credenciales incorrectas
- Agregar producto al carrito
- Flujo completo de compra (checkout)

## Tecnologías usadas
- Playwright
- JavaScript
- Page Object Model (POM)

## Cómo correr los tests
npm install
npx playwright test
